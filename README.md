shooting-gallery
================

Python shooting gallery

Instalace

    conda install -c tlatorre pygame opencv



Kamera a papírový terč
---

* Nalezení terče v obraze
* Výpis skóre
* Generování zvuku



Kamera a projektor
---

* PyGame
* Promítání obrazu terče pomocí projektoru
* Možnost pohyblivého terče
